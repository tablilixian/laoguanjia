# 本地数据存储功能文档

## 1. 功能概述

本功能为「老管家」应用添加了本地数据存储能力，支持：

- **AI 聊天记录本地存储**：自动保存聊天内容，支持历史消息持久化
- **宠物互动日志本地存储**：记录与宠物的所有互动操作
- **增量写入机制**：使用 JSONL 格式，每次只追加新记录，避免性能问题
- **导出功能**：支持将数据导出为标准 JSON 格式，便于备份和分享
- **导入功能**：支持从 JSON 文件导入数据，实现数据迁移

---

## 2. 技术实现

### 2.1 存储位置

```
Documents/home_manager_data/
├── chats_2026-03.jsonl          # AI聊天记录（按月份分文件）
├── chats_2026-04.jsonl
├── pet_interactions_2026-03.jsonl  # 宠物互动日志
├── pet_interactions_2026-04.jsonl
└── exports/                     # 导出的文件目录
    ├── chats_export_2026-03-11.json
    └── pet_logs_export_2026-03-11.json
```

### 2.2 文件格式

**聊天记录 (JSONL 格式)**
```json
{"id":"uuid-xxx","content":"你好","isUser":true,"timestamp":"2026-03-11T10:30:00.000Z"}
{"id":"uuid-yyy","content":"你好！有什么可以帮你的？","isUser":false,"timestamp":"2026-03-11T10:30:01.000Z"}
```

**宠物互动日志 (JSONL 格式)**
```json
{"id":"uuid-xxx","petId":"pet-123","type":"feed","value":20,"createdAt":"2026-03-11T10:30:00.000Z"}
{"id":"uuid-yyy","petId":"pet-123","type":"play","value":20,"createdAt":"2026-03-11T10:35:00.000Z"}
```

**导出文件 (JSON 数组格式)**
```json
{
  "exportDate": "2026-03-11T10:00:00.000Z",
  "totalMessages": 150,
  "messages": [
    {"id":"uuid-xxx","content":"你好","isUser":true,"timestamp":"..."}
  ]
}
```

### 2.3 核心模块

| 模块 | 路径 | 功能 |
|------|------|------|
| LocalStorageService | `lib/core/services/local_storage_service.dart` | 基础文件 I/O 操作 |

| PetInteractionLocalStorage | `lib/core/services/pet_local_storage.dart` | 宠物互动日志存储 |

## 3. 使用说明

### 3.1 入口

1. 打开应用，进入 **设置** 页面
2. 找到 **数据管理** 选项，点击进入

### 3.2 导出聊天记录

1. 在数据管理页面，点击 **「导出聊天记录」**
2. 系统自动导出到 `exports` 目录
3. 导出路径自动复制到剪贴板，可粘贴到文件管理器访问

### 3.3 导入聊天记录

1. 点击 **「导入聊天记录」**
2. 在弹出的对话框中粘贴导出的 JSON 内容
3. 点击「导入」按钮
4. 系统显示导入结果（成功导入的记录数量）

### 3.4 宠物日志操作

与聊天记录操作相同，分别使用「导出宠物日志」和「导入宠物日志」功能。

### 3.5 清空数据

1. 点击 **「清空本地数据」**
2. 在确认对话框中点击「清空」
3. 所有本地存储的聊天记录和宠物日志将被删除

> ⚠️ 注意：此操作不可恢复！

## 6. 常见问题

### Q1: 导出的文件在哪里？
A: 默认导出到应用的` 目录。 `exports路径会在导出成功后显示并复制到剪贴板。

### Q2: 导入需要什么格式？
A: 导入时需要粘贴导出的 JSON 内容（数组格式）。建议使用「导出」功能生成的文件。

### Q3: 数据会丢失吗？
A: 本地数据存储在设备的 Documents 目录下，只要不手动删除或清除应用数据，不会丢失。

### Q4: 云端数据会同步吗？
A: 当前版本仅支持本地存储。宠物互动记录会同时保存到云端（Supabase），但聊天记录仅本地保存。
