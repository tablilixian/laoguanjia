## 二、核心需求

| 需求 | 说明 |
|------|------|
| 语音/文字查询天气 | 用户说"今天天气怎么样" |
| AI 整合 | 用 AI 生成自然语言播报 |
| TTS 语音播放 | 自动朗读天气信息 |
| 位置管理 | 支持设置默认城市 + GPS 自动定位 |
| 智能建议 | AI 根据天气参数给出贴心建议 |

## 四、数据模型

### 4.1 天气偏好

```dart
class WeatherPreference {
  final String? defaultCity;      // 默认城市
  final String? countryCode;     // 国家代码，默认 "CN"
  final bool useGps;             // 是否使用 GPS
  final bool useCelsius;         // 是否使用摄氏度
}
```

### 4.2 天气数据

```dart
class WeatherData {
  final String city;              // 城市名称
  final String country;           // 国家
  final double temperature;       // 温度
  final double feelsLike;         // 体感温度
  final int humidity;            // 湿度
  final String description;       // 天气描述
  final String icon;              // 天气图标代码
  final double windSpeed;         // 风速
  final int windDeg;              // 风向角度
  final int cloudiness;           // 云量
  final int visibility;           // 能见度
  final DateTime sunrise;         // 日出
  final DateTime sunset;          // 日落
  final DateTime dt;              // 数据更新时间
  final double? tempMin;          // 最低温度
  final double? tempMax;          // 最高温度
  final int? pressure;            // 气压
  final int? seaLevel;            // 海平面气压
  final int? groundLevel;         // 地面气压
}
```

### 4.3 扩展天气数据（可选）

| 字段 | 说明 |
|------|------|
| AQI | 空气质量指数 |
| UV Index | 紫外线指数 |
| precipitation | 降水量 |
| probability | 降雨概率 |

## 五、工作流程

```
用户输入："今天天气怎么样"
    │
    ▼
1. AI Chat 页面检测天气关键词
    │
    ▼
2. 获取用户位置
    ├── 优先使用 GPS（如果开启）
    └── 否则使用默认城市
    │
    ▼
3. 调用 Weather Service 查询天气
    │
    ▼ 缓存检查（30分钟有效）
    │
    ▼ 失败重试（最多10次，间隔递增）
    │
    ▼
4. 构建天气数据 Prompt
    │
    ▼
5. 调用 AI 生成自然语言播报
    │
    ▼
6. TTS 语音播放
```

## 八、重试机制

### 8.1 策略

- **最大重试次数**: 10 次
- **初始间隔**: 1 秒
- **递增因子**: 2 倍
- **最大间隔**: 30 秒

### 8.2 间隔序列

| 次数 | 间隔 |
|------|------|
| 1 | 1s |
| 2 | 2s |
| 3 | 4s |
| 4 | 8s |
| 5 | 16s |
| 6 | 30s |
| 7 | 30s |
| ... | 30s |
| 10 | 30s |

### 8.3 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| API Key 未配置 | 提示用户去设置 |
| 网络错误 | 重试 |
| 配额超限 | 重试等待 |
| 城市不存在 | 提示用户检查城市名 |
| 请求超时 | 重试 |

## 十二、API Key 申请指南

### 12.1 申请步骤

1. 访问 https://openweathermap.org
2. 点击 "Sign In" 或 "Create an Account"
3. 注册账号（建议用 Google 账号）
4. 登录后进入 "My API Keys"
5. 点击 "Create API Key"
6. 复制生成的 API Key

### 12.2 注意事项

- API Key 需要 10-15 分钟生效
- 免费版每分钟 60 次请求
- 建议保存好 API Key

### 12.3 国内访问

- OpenWeatherMap 国内访问速度一般
- 如遇访问问题，可考虑切换到和风天气
