## 1. 功能概述

AI 物品助手是专为「老管家」物品管理系统设计的智能对话入口，通过自然语言处理技术，让用户可以用日常对话的方式管理家庭物品。

### 核心特性

- 🤖 **意图识别**：自动识别用户意图（创建/查询/统计）
- 📝 **自然语言**：支持中文日常表达方式
- 🔍 **多维度查询**：支持按位置、类型、标签等多个维度查询
- 📊 **智能统计**：自动统计分析物品数据

## 2. 入口位置

### 方式一：快捷入口（推荐）

在「物品列表」页面，右下角有一个 🤖 机器人图标按钮，点击即可进入 AI 物品助手。

### 方式二：菜单入口

1. 进入「物品管理」页面
2. 点击右上角菜单（⋮ 或更多）
3. 选择「AI 物品助手」

### 方式三：直接访问

通过路由 `/home/items/ai` 直接访问

## 3. 核心功能

### 3.1 AI 创建物品

#### 功能说明

通过自然语言描述，快速创建物品。支持批量创建，一次对话可以添加多个物品。

#### 工作原理

1. 解析用户输入的自然语言
2. 提取物品名称列表
3. 匹配位置信息（从已有的位置列表中选择）
4. 识别物品类型（从21种预设类型中匹配）
5. 自动创建物品记录

#### 支持的句式

| 句式 | 示例 |
|------|------|
| 「帮我把X保存到Y」 | 「帮我把冰箱和电视保存到客厅」 |
| 「把X录入到Y」 | 「把新买的吸尘器录入到储藏室」 |
| 「记录X到Y」 | 「记录这把椅子到书房」 |
| 「新增X放在Y」 | 「新增一盆绿植放在阳台」 |

#### 注意事项

- 位置必须已存在于系统中（可在「位置管理」中添加）
- 类型会自动根据物品名称推断
- 批量创建时用「和」「、」「跟」分隔物品

#### 错误处理

| 情况 | AI 响应 |
|------|----------|
| 未指定位置 | 「抱歉，我没有理解您要把物品保存到哪里」 |
| 位置不存在 | 「抱歉，我没有找到位置'XXX'，请先添加这个位置」 |
| 解析失败 | 「抱歉，我没有理解您要保存的物品，请尝试说'帮我把电视保存到客厅'」 |

### 3.2 AI 查询物品

#### 功能说明

通过自然语言查询物品，支持多种查询维度和模糊匹配。

#### 查询维度

| 维度 | 关键词 | 示例 |
|------|--------|------|
| **按位置** | 卧室、厨房、客厅等 | 「卧室有哪些物品」<br>「厨房里有什么」 |
| **按类型** | 家电、衣物、家具等 | 「有哪些家电」<br>「我的衣物有哪些」 |
| **按标签** | 标签名称 | 「哪些物品打了'需要维修'标签」<br>「有'全新'标签的物品」 |
| **按标签分类** | 状态、价值、季节等 | 「有'高价值'标签的物品」<br>「哪些物品需要维修」 |

#### 模糊匹配支持

AI 助手支持智能模糊匹配，无需精确输入：

| 用户输入 | 匹配结果 |
|----------|----------|
| 「卧室」 | 主卧、次卧、儿童房等 |
| 「客厅」 | 客厅、起居室 |
| 「浴室」 | 浴室、卫生间、洗手间 |
| 「厨房」 | 厨房 |

#### 查询特定物品

| 句式 | 示例 |
|------|------|
| 「我有哪些X」 | 「我有哪些衣服」 |
| 「找出所有的X」 | 「找出所有的家电」 |
| 「查看X的物品」 | 「查看厨房的物品」 |

---

### 3.3 AI 智能统计

#### 功能说明

自动统计分析物品数据，提供全面的物品统计报告。

#### 支持的统计项

| 统计项 | 关键词 | 示例 |
|--------|--------|------|
| **物品总数** | 多少、有多少 | 「我家有多少物品」<br>「总共有多少东西」 |
| **按类型分布** | 按类型统计、分类统计 | 「按类型统计一下」<br>「各类物品数量」 |
| **总价值** | 价值、总价、值多少钱 | 「物品总价值是多少」<br>「加起来值多少钱」 |
| **保修状态** | 保修、到期、快到期 | 「哪些保修快到期了」<br>「保修状态」 |
| **过期保修** | 过期、失效 | 「已过期的保修有哪些」 |

#### 统计报告示例

```
📊 **物品统计**

• 总物品数: 42 件

**按类型分布:**
  🔌 家电: 12 件
  👕 衣物: 8 件
  🛋️ 家具: 6 件
  ...

**💰 总价值: ¥125,800.00**

**⚠️ 即将到期的保修 (30天内): 3 件**
  • 美的空调 (2026-04-15)
  • 西门子冰箱 (2026-04-20)
  • 小米电视 (2026-04-28)

**❌ 已过期的保修: 5 件**
```

## 5. 常见问题

### Q1: 为什么 AI 创建物品失败了？

**可能原因：**
1. 位置不存在 → 请先在「位置管理」中添加
2. API Key 未配置 → 请在「设置 → AI 设置」中配置
3. 网络问题 → 检查网络连接

### Q2: 为什么查询结果不对？

**可能原因：**
1. 模糊匹配限制 → 尝试更精确的关键词
2. 物品未设置位置 → 检查物品详情中的位置信息
3. 物品未设置标签 → 检查物品详情中的标签信息

### Q3: 如何让 AI 更准确？

**建议：**
1. 使用完整的物品/位置名称
2. 避免歧义表达
3. 多使用示例中的句式

### Q4: 支持语音输入吗？

目前版本暂不支持语音输入，预计后续版本会添加。
